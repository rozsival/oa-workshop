OA TGM React Workshop Sources
=============================

Use https://github.com/creationix/nvm for node.js installation before you start.

Installation
------------
```bash
git clone git@github.com:teekey99/oa-workshop.git
cd oa-workshop
npm install
```

Development
-----------
```bash
npm run start
```

The app is running at http://localhost:3000.
